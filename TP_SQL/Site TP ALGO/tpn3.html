TP N°3 Les Requêtes statistiques mono table

  • 1. Compter le nombre de chambres total.
    -------
    20
    sjt=# SELECT COUNT (CHB_ID) FROM T_CHAMBRE;
  • 2. Retrouver le prix maximal d’une chambre.
    --------
    512.00
    sjt=# SELECT MAX (TRF_CHB_PRIX) FROM TJ_TRF_CHB;
  • 3. Retrouver le nombre de places totales dans l'hôtel.
    -----54
    sjt=# SELECT SUM(CHB_COUCHAGE) FROM T_CHAMBRE;
  • 4. Retrouver le prix moyen des chambres.
    ----------------------
    326.3700000000000000
    sjt=# SELECT AVG (TRF_CHB_PRIX) FROM TJ_TRF_CHB;
  • 5. Compter le nombre de clients dont l’enseigne est renseignée.
    -------
    19
    sjt=# SELECT COUNT (CLI_ID) FROM T_CLIENT WHERE CLI_ENSEIGNE IS NOT NULL;
  • 6. Retrouver le prix moyen des chambres durant la période du 15/01/2000 au 15/06/2000.
    ⚠ Pas de jointure ici.
    Ne retourne rien car il n’y a pas de chambres disponibles dans cet intervalle de dates.
    sjt=# SELECT AVG (TRF_CHB_PRIX) FROM TJ_TRF_CHB WHERE TRF_DATE_DEBUT BETWEEN '15/01/2000' AND '15/06/2000';
  • 7. Compter le nombre de clients par genre. (Nombre de M., de Mme. et de Melle. en une seule requête)
    ---------+-------
    M. | 83
    Melle. | 2
    Mme. | 15
    (3 rows)
    sjt=# SELECT TIT_CODE, COUNT (CLI_ID) FROM T_CLIENT GROUP BY TIT_CODE;
  • 8. Compter le nombre de chambres par étage.
    ----------+-------
    RDC | 4
    1er | 8
    2e | 8
    (3 rows)
    sjt=# SELECT CHB_ETAGE, COUNT (CHB_ID) FROM T_CHAMBRE GROUP BY CHB_ETAGE;
  • 9. Retrouver le nombre de places totales dans l'hôtel par étage.
    ----------+-----
    RDC | 9
    1er | 23
    2e | 22
    (3 rows)
    sjt=# SELECT CHB_ETAGE, SUM (CHB_COUCHAGE) FROM T_CHAMBRE GROUP BY CHB_ETAGE;
  • 10. Retrouver le nombre de places totales dans l'hôtel par étage seulement si ce nombre est supérieur à 15.
    ----------+-----
    1er | 23
    2e | 22
    (2 rows)
    sjt=# SELECT CHB_ETAGE, SUM (CHB_COUCHAGE) FROM T_CHAMBRE GROUP BY CHB_ETAGE HAVING SUM(CHB_COUCHAGE) >15;
